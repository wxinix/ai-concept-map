# AI Concept Map

An interactive visual guide to the building blocks of modern AI. A single self-contained HTML file -- no build tools, no frameworks, no dependencies.

**[Live Demo](https://wxinix.github.io/ai-concept-map/)**

## Overview

AI Concept Map organizes key AI concepts into five layers, from foundational primitives to emerging frontiers:

| Layer | Name | Examples |
|-------|------|----------|
| 01 | **Primitives** | Prompt, LLM, Transformer, Embedding, Vector Space |
| 02 | **Compositions** | Function Calling, MCP, RAG, Vector Database, Guardrails |
| 03 | **Model Adaptation** | SFT, LoRA, RLHF / DPO, Knowledge Distillation |
| 04 | **Deployment** | Agent, Fine-tuning, Framework, Evals, Red Teaming |
| 05 | **Emerging** | Multi-Agent Systems, Thinking Models, Interpretability |

Click any concept to read a concise explanation of what it is, why it matters, and how it connects to the rest of the stack.

## Features

- **Single file** -- just open `index.html` in a browser
- **Two-column layout** -- concept map on the left, explanation sidebar on the right
- **Resizable split** -- drag the divider to adjust column widths
- **Collapsible layers** -- click any layer header to expand or collapse
- **Dark / light mode** -- toggle in the top-right corner (respects system preference)
- **Responsive** -- stacks vertically on mobile screens

## Usage

```
git clone https://github.com/wxinix/ai-concept-map.git
cd ai-concept-map
open index.html
```

Or simply download `index.html` and open it in any modern browser.

## Why This Exists

Every AI product is a combination of a small set of building blocks. This map helps you place any pitch, architecture, or product into context -- whether you are learning AI concepts for the first time or evaluating a new tool.
